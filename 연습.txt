객체 : 현실 세상에 존재하는 유형 또는 무형(생각, 개념)의 모든 것을 공통적인 특징만 가지고 
컴퓨터 세상으로 옮긴 것 

속성: 공통된 특징
기능: 객체의 특징을 변경시키는 것

클래스 : 한 종류의 객체를 여러개 만들어 줄 틀

변수: 데이터를 저장하기 위한 메모리 공간의 이름입니다.

조건문:프로그램의 실행 흐름을 조절하기 위해 사용됩니다.

주로 if, else if, else 키워드를 사용합니다.

반복문 :동일한 코드 블록을 여러 번 실행할 때 사용됩니다.

for, while, 등이 있습니다.

배열: 같은 타입의 형식의 여러 값을 저장하는 자료 구조입니다.
